tegenaria是一个基于golang开发的快速、高效率的网络爬虫框架

# 特性

- 支持分布式  

- 支持自定义分布式组件  

- 支持自定义的事件监控

- 支持命令行控制
  
  ## 安装
1. go 版本要求>1.17  

```bash
go get -u github.com/wetrycode/tegenaria
```

2. 在您的项目中导入

```go
import "github.com/wetrycode/tegenaria"
```
