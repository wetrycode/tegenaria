tegenaria是一个基于golang开发的快速、高效率的网络爬虫框架,更多详细内容参见[快速开始](quickstart.md)。  

# 特性

- 支持分布式  

- 支持自定义分布式组件,包括去重、request缓存队列和基本统计指标的分布式运行    

- 支持自定义的事件监控

- 支持命令行控制  

- 支持gRPC远程控制  

- ~~支持定时轮询启动爬虫~~
